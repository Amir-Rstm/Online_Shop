A simple e-commerce system implementation to demonstrate Object-Oriented Programming (OOP) concepts in Java.

Features
- **Product Management:** Abstract base class for different product types.
- **Polymorphism:** Handling various products (Electronics, Clothing) in a single Cart.
- **Shopping Cart:** Add items, calculate total cost with specific tax/profit rules.
- **Receipt Generation:** Using Downcasting to print specific product details (Size, Warranty).

Concepts Used
- **Abstraction** (Abstract Classes & Methods)
- **Inheritance** (`extends`)
- **Interfaces** (`implements Returnable`)
- **Polymorphism** (Method Overriding)
- **Encapsulation** (Private fields, Getters/Setters)
- **Downcasting & instanceof**
---
*Created as a learning project to master Java Core concepts.*
